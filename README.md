# 移动软件开发实验代码整合仓库

## 课程信息

课程以**移动应用开发基本编程能力**培养为目标，突出小程序基础知识的讲授，并结合实践案例及项目，使学习者从无到有，逐步掌握微信小程序的基本开发技能，为微信小程序的高级开发打下坚实基础。

## 课程大致安排

- 课堂教学：研讨性教学、讨论、项目汇报
- 课后作业：在雨课堂里提交作业
- 项目实践：实验（3周）、个人项目（4周）
- 成绩构成：个人实验60% + 个人项目40% + 奖励成绩 - 惩罚成绩 - 缺勤
